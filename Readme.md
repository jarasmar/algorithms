# ALGORITHM COMPLEXITY

### SHUFFLING
Mix all the elements in an array to get a random order

```
INPUT        | OUTPUT
[1]          | [1]
[1, 2]       | [1,2] or [2, 1]
[1, 1, 3]    | [1, 3, 1] or [3, 1, 1] or [1, 1, 3]
[]           | error
```

### REVERSE
Reverse the order of the elements in an array

```
INPUT        | OUTPUT
[1]          | [1]
[1, 2]       | [2, 1]
[1, 2, 3]    | [3, 2, 1]
[]           | error
```
